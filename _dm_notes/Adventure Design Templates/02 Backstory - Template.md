# Adventure Backstory Template

## How to use this note

Use this note for the world context behind the adventure. It does not need to follow the same shape every time. Write whatever helps: a short lore essay, a timeline, faction notes, rumors, unresolved mysteries, or the hidden truth behind events. Keep this separate from runnable scene prep, link the source notes you rely on, and mark uncertain or speculative material clearly.

## Sources and anchors

Existing notes, facts, places, people, groups, events, or mysteries this adventure depends on.

## Backstory

Write the relevant context in whatever form fits the adventure.

## What might matter in play

Notes from the backstory that could become clues, NPC motives, locations, constraints, stakes, consequences, or complications.

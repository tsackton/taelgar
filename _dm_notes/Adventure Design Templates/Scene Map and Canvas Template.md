# Scene Map and Canvas Template

## How to use this note

Use this note alongside an Obsidian Canvas. The Canvas is the playable map of the adventure: scene cards, text cards, groups, and labeled connections. This note records the conventions and keeps a simple text index so the Canvas does not become the only source of orientation.

## Card types

- Scene note card: a substantial playable scene with its own note.
- Text card: a small beat, transition, reminder, unresolved idea, or optional moment.
- Group: an act, nested site, dungeon, investigation cluster, faction thread, or other substructure.

## Color legend

- Green: start or hook.
- Blue: main adventure spine.
- Gray: optional scene.
- Purple: nested site, dungeon, or inner adventure.
- Red: climax.
- Yellow: draft, unresolved, or needs design work.

## Line labels

Use labels that explain why one scene leads to another.

- Good: follows blood trail
- Good: captures cultist
- Good: opens sealed crypt
- Weak: then
- Weak: next

# Adventure Scene Template

## How to use this note

Use this note for a substantial scene on the adventure Canvas. Keep it freeform. Start with only the pieces that help you run the scene, then add detail as needed. A social scene, travel scene, combat scene, vision, puzzle, and dungeon room do not need the same structure.

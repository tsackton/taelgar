# Adventure Backstory Template

## How to use this note

Use this note to explain why the adventure exists in Taelgar. This is the world-context source of truth, not the runnable adventure text. Write enough history, faction logic, and hidden truth that the adventure feels grounded. Then translate that backstory into things that can actually appear in play.

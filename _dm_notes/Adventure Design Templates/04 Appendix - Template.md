# Adventure Appendix Template

## How to use this note

Use this for support material that should not live inside one scene note. Keep it scan-friendly. If an appendix grows large, split it into a focused note such as NPCs, Monsters, Treasure, Handouts, Rules, or Names.

Can link out to material in Extras, which can function like an adventure-specific staging area; adventure assets; and anything else useful to store as adventure-related notes.

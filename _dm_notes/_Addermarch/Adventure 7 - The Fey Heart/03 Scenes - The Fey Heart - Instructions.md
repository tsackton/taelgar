# Scenes - Instructions

## How to use this note

Use this as the orientation note for building the adventure's scene structure. The actual structure should live in a Canvas, not in a table. This page is mostly instructions and conventions; only add a scene list or notes here if they help keep the Canvas understandable.

## Basic setup

Make one Canvas for the adventure's scene map. Use it to show the playable structure: how scenes connect, where the critical path runs, what is optional, where the climax sits, and where a scene expands into a nested location or inner adventure.

Do not make every idea into a note. Use Canvas text cards for small beats, reminders, transitions, possible complications, or undeveloped ideas. Create a separate scene note only when the scene needs enough detail to run or revise independently.

## Canvas conventions

- Scene note card: substantial playable scene with its own note.
- Text card: small beat, transition, reminder, option, or draft idea.
- Group: act, chapter, dungeon, investigation cluster, social hub, faction thread, or nested site.
- Green: start or hook.
- Blue: main spine.
- Gray: optional scene.
- Purple: nested site or inner adventure.
- Red: climax.
- Yellow: draft or unresolved.

## Connections

Label lines with the reason the next scene becomes available. Prefer conditions, discoveries, decisions, or consequences over vague sequence labels.

Useful labels:

- follows the blood trail
- captures the cultist
- finds the abbey records
- opens the sealed crypt
- fails to stop the ritual

Weak labels:

- then
- next
- after that

## Nested scenes

Some scenes are really small adventures. A dungeon, heist, investigation hub, wilderness route, or complex social event can appear as one card on the parent Canvas and then have its own internal Canvas, map, or note.

Use a nested structure when the parent Canvas would become cluttered or when the scene has its own goal, challenge, routes, and outcomes.

## Scene notes

Scene notes should stay freeform. Let the scene decide the format. A negotiation, chase, dungeon room, travel sequence, ritual, and climactic battle need different prep.

Useful things to include when relevant:

- purpose
- entry
- opening frame
- location
- cast
- what is really happening
- player choices
- obstacles
- clues or revelations
- timer or pressure
- outcomes
- exits

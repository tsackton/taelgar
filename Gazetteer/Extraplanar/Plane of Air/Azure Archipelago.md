# The Azure Archipelago
>[!info]+ Information  
> `$=dv.view("_scripts/view/get_Affiliations")`  
>> `$=dv.view("_scripts/view/get_Whereabouts")`

The Azure Archipelago is a cluster of floating islands in the [[Elemental Plane of Air]]. It is described in [[The Lay of Estar]] and in later reports as a set of lush, verdant islands drifting in a synchronized dance, surrounded by walls of perpetual mist.
